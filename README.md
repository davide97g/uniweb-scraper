# uniweb-scraper
Uniweb python scraper.

---
Exposes two simple APIs:
- `exams/` : return the list of current exams
- `exams/update` : update the current status of the exams scraping the website  

## Project setup
- create an `heroku` project
- create a `mongodb cloud atlas` project
- link `github` repository to `heroku` deployment system (optional)
- run `setup.py` in order to create local `.env` file _(you will need `heroku` and `mongodb cloud atlas` configured)_

## MongoDB setup
- create a [mongodb cloud atlas](https://www.mongodb.com/cloud/atlas) _project_
- create a new _cluster_
- create a new _database_ called `university`
- create a new _collection_ called `exams`

## Heroku setup
- create a project on [heroku](https://www.heroku.com/)

### Heroku buildpacks
- `https://github.com/heroku/heroku-buildpack-google-chrome`
- `https://github.com/heroku/heroku-buildpack-chromedriver`

### Configuration variables
- `CHROMEDRIVER_PATH /app/.chromedriver/bin/chromedriver`
- `GOOGLE_CHROME_BIN /app/.apt/usr/bin/google-chrome`
- all the variables inside the `.env` file generated by `setup.py`
